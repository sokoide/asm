# 8086 アセンブラ DOS .COM ワークショップ

DOS 向け 16-bit .COM プログラムを段階的に学ぶ。
NASM（`-f bin`）で直接バイナリを生成し、DOSBox で実行する。

## 前提条件

| ツール | 用途             | インストール例       |
| :---   | :---             | :---                 |
| NASM   | アセンブラ       | `apt install nasm`   |
| DOSBox | DOS エミュレータ | `apt install dosbox` |

## ビルドと実行

```bash
# ビルド
nasm -f bin hello.asm -o hello.com

# DOSBox で実行
dosbox
```

DOSBox 内で実行:
```bash
c:
mount c ~/tmp
c:
hello.com
```

## .COM と .EXE の違い

| 項目             | .COM                   | .EXE               |
| :---             | :---                   | :---               |
| セグメント       | 単一セグメント（64KB） | 複数セグメント可能 |
| エントリポイント | ORG 0x100 から固定     | ヘッダ指定可       |
| 最大サイズ       | 64KB - 256B            | 最大 1MB           |
| ヘッダ           | なし                   | MZ ヘッダあり      |
| DS 初期値        | CS と同じ              | PSP セグメント     |

## .COM フォーマットの基礎

```text
┌────────────────────┐
| PSP (256 バイト)   | DOS が管理するプログラム情報
├────────────────────┤
| コード (最大 64KB  | ORG 0x100 から開始
└────────────────────┘
```

- **ORG 0x100**: コードが 256 バイト目から始まる
- **単一セグメント**: .COM ファイル全体が 1 つのセグメント内に収まる
- **DS = CS**: 起動時に DS は CS と同じセグメントを指す
- **最大 64KB**: プログラム全体が 64KB を超えない

## hello.asm 解説

```assembly
org 0x100              ; .COM ファイルのエントリポイント（256 バイト目）

start:
    mov ah, 09h        ; DOS 呼び出し番号 09（文字列出力）
    lea dx, message    ; メッセージのアドレスを DS:DX に設定
    int 21h            ; DOS 割り込みを呼び出し
    
    mov ah, 4ch        ; DOS 呼び出し番号 4Ch（プログラム終了）
    int 21h            ; DOS 割り込みを呼び出し

message db "Hello, COM world!$"  ; '$' で終端される文字列
```

## 8086 命令リファレンス

### 基本命令

| 命令           | 説明                 |
| :---           | :---                 |
| `MOV dst, src` | データ転送           |
| `ADD dst, src` | 加算                 |
| `SUB dst, src` | 減算                 |
| `INC dst`      | インクリメント       |
| `DEC dst`      | デクリメント         |
| `PUSH src`     | スタックへ保存       |
| `POP dst`      | スタックから復元     |

| `RET`          | サブルーチンから戻る |

## DOS 割り込みリファレンス

### INT 21h — DOS サービス

| AH    | 機能                   | 入力                   | 出力       |
| :---  | :---                   | :---                   | :---       |
| `01h` | 文字入力（エコーあり） | -                      | AL = ASCII |
| `02h` | 文字出力               | DL = 文字              | -          |
| `09h` | 文字列出力             | DS:DX → '$' 終端文字列 | -          |
| `4Ch` | プログラム終了         | AL = リターンコード    | -          |

## ビルド手順

1. **アセンブル**:
   ```bash
   nasm -f bin hello.asm -o hello.com
   ```

2. **実行環境準備**:
   ```bash
   # DOSBox 起動後
   mount c ~/tmp
   c:
   ```

3. **実行**:
   ```bash
   hello.com
   ```

4. **逆アセンブル（デバッグ用）**:
   ```bash
   ndisasm -b 16 hello.com
   ```
