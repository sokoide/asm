# PowerPC アセンブラ ワークショップ

PowerPC bare-metal（空芯）アセンブラの基礎から応用まで、12のシナリオで段階的に学ぶ。

## 前提条件

| ツール | 用途                 | インストール例                |
| :---   | :---                 | :---                          |
| Clang  | PowerPC アセンブラ   | `apt install clang`           |
| LLD    | PowerPC リンカ       | `apt install lld`             |
| QEMU   | PowerPC エミュレータ | `apt install qemu-system-ppc` |

## ビルドと実行

```bash
# 全シナリオをビルド
make

# 特定のシナリオを QEMU で実行
make run S=s01_hello

# 全シナリオを順に実行
make runall

# 逆アセンブル
make dump S=s01_hello

# 掃除
make clean
```

## シナリオ一覧

### S01: Hello World

**難易度**: ★☆☆☆☆

最も基本的なPowerPC bare-metalプログラム。UART経由で文字列を表示して停止する。

| 学習項目       | 説明                                                       |
| :---           | :---                                                       |
| スタック初期化 | `lis %r1, 0x0000; ori %r1, %r1, 0x1000` でR1を0x1000に設定 |
| LIS/ORI        | 32bit定数を組み立てる（High/Lowアドレス）                  |
| STB            | メモリへのバイト書き込み（UART出力用）                     |
| BL/BLR         | サブルーチン呼び出しと戻り                                 |
| 16550 UART     | 0xEF600300へ文字出力（LSR bit 5待ち）                      |

## PowerPC 命令リファレンス

### データ転送

| 命令                 | 動作                                 |
| :---                 | :---                                 |
| `lis rd, imm16`      | rd = imm16 << 16（上位16ビット設定） |
| `ori rd, ra, imm16`  | rd = ra                              | imm16（下位16ビット設定） |
| `li rd, imm16`       | rd = 0xFFFF                          | imm16（16bit即値設定）    |
| `addi rd, ra, imm16` | rd = ra + imm16（即値加算）          |
| `stb rs, d(ra)`      | メモリにバイト書き込み               |
| `lbz rd, d(ra)`      | メモリからバイト読み込み             |
| `stwu rs, d(ra)`     | 書き込みとra更新（push用）           |
| `lbzu rd, d(ra)`     | 読み込みとra更新（auto-increment）   |
| `stw rs, d(ra)`      | メモリにワード書き込み               |
| `lwz rd, d(ra)`      | メモリからワード読み込み             |
| `mr rd, ra`          | rd = ra（レジスタコピー）            |

### 算術

| 命令                 | 動作                           |
| :---                 | :---                           |
| `add rd, ra, rb`     | rd = ra + rb                   |
| `addi rd, ra, imm`   | rd = ra + imm（即値加算）      |
| `subf rd, ra, rb`    | rd = rb - ra（減算、注意順序） |
| `mullw rd, ra, rb`   | rd = ra × rb（32bit乗算）      |
| `divw rd, ra, rb`    | rd = ra ÷ rb（32bit除算）      |
| `addic. rd, ra, imm` | 加算+条件レジスタ更新          |
| `neg rd, ra`         | rd = -ra（2の補数）            |

### 論理

| 命令                | 動作                         |
| :---                | :---                         |
| `and rd, ra, rb`    | rd = ra & rb（論理積）       |
| `andi. rd, ra, imm` | rd = ra & imm（即値AND）     |
| `or rd, ra, rb`     | rd = ra                      | rb（論理和）  |
| `ori rd, ra, imm`   | rd = ra                      | imm（即値OR） |
| `xor rd, ra, rb`    | rd = ra ^ rb（排他的論理和） |
| `xori rd, ra, imm`  | rd = ra ^ imm（即値XOR）     |
| `nor rd, ra, rb`    | rd = ~(ra                    | rb）          |
| `not rd, ra`        | rd = ~ra（ビット反転）       |

### シフト

| 命令                        | 動作                              |
| :---                        | :---                              |
| `slw rd, ra, rb`            | rd = ra << rb（左論理シフト）     |
| `slwi rd, ra, sh`           | rd = ra << sh（左即値シフト）     |
| `srw rd, ra, rb`            | rd = ra >> rb（右論理シフト）     |
| `srwi rd, ra, sh`           | rd = ra >> sh（右即値シフト）     |
| `sraw rd, ra, rb`           | rd = ra >> rb（右算術シフト）     |
| `srawi rd, ra, sh`          | rd = ra >> sh（右即値算術シフト） |
| `rlwinm rd, ra, sh, mb, me` | 回転+マスク（32bit操作）          |

### 比較

| 命令             | 動作                   |
| :---             | :---                   |
| `cmpw ra, rb`    | ra - rbでフラグ設定    |
| `cmpwi ra, imm`  | ra - immでフラグ設定   |
| `cmplw ra, rb`   | ra - rbで符号付き比較  |
| `cmplwi ra, imm` | ra - immで符号付き比較 |
| `test rd, ra`    | rd & raでフラグ設定    |

### 分岐

| 命令         | 動作                   | 条件                     |
| :---         | :---                   | :---                     |
| `b label`    | 無条件ジャンプ         | —                        |
| `beq label`  | 等しい場合             | CR0.ZF=1                 |
| `bne label`  | 等しくない場合         | CR0.ZF=0                 |
| `blt label`  | より小さい（符号付き） | CR0.SF≠OF                |
| `ble label`  | 以下（符号付き）       | CR0.ZF=1 or SF≠OF        |
| `bgt label`  | より大きい（符号付き） | CR0.ZF=0 and SF=OF       |
| `bge label`  | 以上（符号付き）       | CR0.ZF=1 or SF=OF        |
| `bltl label` | より小さい（符号なし） | CR0.CF=1                 |
| `blel label` | 以下（符号なし）       | CR0.CF=1 or ZF=1         |
| `bgtl label` | より大きい（符号なし） | CR0.CF=0 and ZF=0        |
| `bgel label` | 以上（符号なし）       | CR0.CF=0                 |
| `bl label`   | リンク付きジャンプ     | LRにリターンアドレス保存 |

### スタック

| 命令             | 動作                     |
| :---             | :---                     |
| `stwu rs, d(r1)` | 書き込みとR1更新（push） |
| `lwz rd, d(r1)`  | 読み込み（pop）          |
| `addi r1, r1, d` | R1を調整（スタック操作） |

### サブルーチン

| 命令       | 動作                       |
| :---       | :---                       |
| `bl label` | リンク付きジャンプ         |
| `blr`      | リンクレジスタへジャンプ   |
| `mflr rd`  | リンクレジスタから読み込み |
| `mtlr rs`  | リンクレジスタに書き込み   |

### 制御

| 命令          | 動作                     |
| :---          | :---                     |
| `nop`         | 何もしない（nop）        |
| `tw ra, rb`   | トリップ命令（例外発生） |
| `twi ra, imm` | 即値トリップ命令         |

## ハードウェア情報

### QEMU bamboo マシン

- **CPU**: PowerPC 405GP
- **メモリ**: 64MB RAM
- **UART**: 16550 compatible at 0xEF600300
- **タイマ**: Time Base register（特殊レジスタ）

### 16550 UART レジスタ

| オフセット | 名称 | 説明                                    |
| :---       | :--- | :---                                    |
| 0x00       | THR  | Transmit Holding Register（書き込み用） |
| 0x05       | LSR  | Line Status Register（読み込み用）      |
- **LSR bit 5**: THRE（Transmitter Holding Register Empty）- 書き込み可能フラグ
- **LSR bit 0**: DR（Data Ready）- 受信データありフラグ
