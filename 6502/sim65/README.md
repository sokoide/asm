# 6502 アセンブラ ワークショップ

cc65 付属のシミュレータ sim65 を使い、12 のシナリオで 6502 アセンブリを段階的に学ぶ。

## 前提条件

| ツール      | 用途                | インストール例                           |
| :---        | :---                | :---                                     |
| cc65 (cl65) | アセンブラ + リンカ | `apt install cc65` / `brew install cc65` |
| sim65       | 6502 シミュレータ   | cc65 に同梱                              |

## ビルドと実行

```bash
# 全シナリオをビルド
make

# 特定のシナリオを実行
make run S=s01_hello

# 掃除
make clean
```

## シナリオ一覧

### S01: Hello World

**難易度**: ★☆☆☆☆

cc65 ランタイムを使った最小限のプログラム。`_puts` で文字列を出力する。

| 学習項目                    | 説明                                        |
| :---                        | :---                                        |
| `.import` / `.export`       | 外部シンボルのインポート・エクスポート      |
| `.segment`                  | RODATA / CODE セグメントの使い分け          |
| `lda #<addr` / `ldx #>addr` | 16 ビットアドレスの下位・上位バイトをロード |
| `jsr _puts`                 | C ランタイムの文字列出力関数                |
| `.asciiz`                   | ヌル終端文字列の定義                        |
| `rts`                       | C の `main` から `return 0` として終了      |

### S06: アドレッシングモード

**難易度**: ★★★☆☆

6502 の全アドレッシングモードを網羅的に体験する。

| 学習項目        | 説明                                       |
| :---            | :---                                       |
| 即値 (`#`)      | `LDA #$42` — 定数を直接指定                |
| ゼロページ      | `LDA $20` — $00-$FF への短いアクセス       |
| 絶対アドレス    | `LDA addr` — 16 ビット絶対アドレス         |
| ゼロページ,X    | `LDA $20,X` — インデックス付き ZP          |
| 絶対,X / 絶対,Y | `LDA addr,X` / `LDA addr,Y`                |
| (Indirect,X)    | `LDA ($20,X)` — ZP ポインタ + X の事前加算 |
| (Indirect),Y    | `LDA ($20),Y` — ZP ポインタ + Y の事後加算 |
| インプライド    | `TAX` — 暗黙のオペランド                   |
| アキュムレータ  | `ASL A` — A レジスタを直接操作             |
| 相対            | `BEQ label` — 分岐命令のオフセット指定     |

## cc65 C ランタイム I/O

cc65 の sim6502 ターゲットは C ランタイムとリンクする。アセンブリから以下の関数を呼び出せる。

| 関数       | 引数                 | 動作                                  |
| :---       | :---                 | :---                                  |
| `_putchar` | A = 文字             | 1 文字を stdout に出力                |
| `_puts`    | A/X = 文字列アドレス | ヌル終端文字列を出力（改行付き）      |
| `_getchar` | なし                 | 1 文字を stdin から入力（A に戻り値） |

**注意**: `_putchar` / `_puts` / `_getchar` は X, Y レジスタを破壊する。ループカウンタとして使用している場合は、呼び出しの前後で必ず退避・復元すること。ゼロページ上の変数も破壊される可能性があるため、固定ゼロページアドレス（`$F0` など高アドレス）を使用するか、ハードウェアスタックで退避すること。

## cc65 アセンブラ疑似命令

| 疑似命令          | 動作                                   |
| :---              | :---                                   |
| `.import sym`     | 外部シンボルのインポート               |
| `.export sym`     | シンボルを外部公開                     |
| `.segment "NAME"` | セグメント指定（CODE/RODATA/DATA/BSS） |
| `.res size`       | 未初期化領域の確保（ゼロページ用）     |
| `.space size`     | 未初期化領域の確保                     |
| `.byte vals`      | バイト列の定義                         |
| `.word vals`      | ワード列の定義                         |
| `.asciiz "str"`   | ヌル終端文字列の定義                   |
| `.addr sym`       | 16-bit アドレス定数                    |
