# PowerPC アセンブラ ワークショップ

PowerPC bare-metal（空芯）アセンブラの基礎から応用まで、12 のシナリオで段階的に学ぶ。

## 前提条件

| ツール | 用途                 | インストール例                |
| :---   | :---                 | :---                          |
| Clang  | PowerPC アセンブラ   | `apt install clang`           |
| LLD    | PowerPC リンカ       | `apt install lld`             |
| QEMU   | PowerPC エミュレータ | `apt install qemu-system-ppc` |

## ビルドと実行

```bash
# 全シナリオをビルド
make

# 特定のシナリオを QEMU で実行
make run S=s01_hello

# 全シナリオを順に実行
make runall

# 逆アセンブル
make dump S=s01_hello

# 掃除
make clean
```

## シナリオ一覧

### S01: Hello World

**難易度**: ★☆☆☆☆

最も基本的な PowerPC bare-metal プログラム。UART 経由で文字列を表示して停止する。

| 学習項目       | 説明                                                       |
| :---           | :---                                                       |
| スタック初期化 | `lis %r1, 0x0000; ori %r1, %r1, 0x1000` でR1を0x1000に設定 |
| LIS/ORI        | 32bit定数を組み立てる（High/Lowアドレス）                  |
| STB            | メモリへのバイト書き込み（UART出力用）                     |
| BL/BLR         | サブルーチン呼び出しと戻り                                 |
| 16550 UART     | 0xEF600300へ文字出力（LSR bit 5待ち）                      |

## ハードウェア情報

### QEMU bamboo マシン

- **CPU**: PowerPC 405GP
- **メモリ**: 64MB RAM
- **UART**: 16550 compatible at 0xEF600300
- **タイマー**: Time Base register（特殊レジスタ）

### 16550 UART レジスタ

| オフセット | 名称 | 説明                                    |
| :---       | :--- | :---                                    |
| 0x00       | THR  | Transmit Holding Register（書き込み用） |
| 0x05       | LSR  | Line Status Register（読み込み用）      |

- **LSR bit 5**: THRE（Transmitter Holding Register Empty）- 書き込み可能フラグ
- **LSR bit 0**: DR（Data Ready）- 受信データありフラグ
